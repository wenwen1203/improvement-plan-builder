# 改进计划制定参考框架

本文件包含各规划阶段的详细质量标准、模板和讨论提示，由 SKILL.md 按需引用。

---

## 一、改进计划关键要素

一个完整的改进计划包含以下八个要素：

| 要素 | 说明 |
|---|---|
| **目标（Goal）** | 学校希望通过改进最终达成的长期结果或总体方向 |
| **具体目标（Objective）** | 为实现Goal需达成的具体、可衡量的短期成果 |
| **行动** | 为实现每个Objective所采取的具体步骤 |
| **时间** | 每个行动的实施时间，含开始和结束日期 |
| **负责人** | 负责每个行动执行和落实的具体人员 |
| **资源需求** | 实现每个行动所需的资源（资金、设备、培训、外部支持等） |
| **成功标准** | 衡量行动和目标达成情况的标准或指标 |
| **复盘与评估** | 对行动实施效果的总结和评估方式 |

---

## 二、逻辑拆解路径

### 从Goal到Objective

Goal提供总体方向，Objective将其分解为具体的、可操作的步骤。

**拆解角度参考**（不是每个都需要，选2-4个最相关的）：

- **教师能力建设**：教师需要掌握什么新策略？（如培训、教研、课堂实践）
- **课堂实践改变**：课堂里需要发生什么不同？（如教学设计、评价方式、学习活动）
- **学生支持体系**：学生需要获得什么额外支持？（如个性化辅导、学习资源）
- **家校协同**：家庭层面需要配合什么？（如家长讲座、家庭学习指导）
- **资源与制度建设**：学校层面需要提供什么保障？（如资源包开发、制度流程）

**Objective的写法要求**：
- 具体且短期：有明确的时间框架和结果
- 可衡量：包含具体的数字或可观察的变化
- 直接服务于Goal：每个Objective都应该能回答"它怎么帮助实现Goal？"

### 从Objective到行动

**行动设计讨论提示**：

- 我们是否需要实施新的策略或框架？
- 我们是否需要创建新的政策或指导？
- 我们是否需要购买或开发新的资源？
- 我们是否需要提供教师培训？
- 我们是否需要在教师专业发展中增加新活动？
- 我们是否需要来自专家的建议？

**行动筛选讨论提示**：

- 谁将负责实施？（思考谁有相关专业能力）
- 在负责教师的时间里，这些行动都能完成吗？
- 是否有其他教师有能力分担？
- 哪些行动对达成目标是**至关重要**的？
- 如果不采取某个行动，我们是否仍然能达成目标？

**行动之间的依赖关系**：
- 有些行动必须先完成才能启动下一个
- 有些行动可以并行推进
- 排序时要考虑学期节奏和教师工作负荷

---

## 三、时间规划讨论提示

对每个行动，引导教师思考：

- 这个行动是否已经是教师的日常工作职责？
- 如果是一次性任务，预计花费多少小时？
- 如果是周期性任务（每周/每学期），每次花费多少时间？
- 需要多人参与吗？涉及多少人？
- 行动之间有依赖关系吗？必须先完成哪个？

---

## 四、资源需求讨论提示

对每个行动，引导教师思考：

- 是否需要外部专家支持？提供什么类型？
- 是否需要特定的培训材料或技术平台？
- 是否需要额外的场地或设备？
- 是否需要骨干教师的协助？
- 学校现有资源能覆盖吗？需要申请额外预算吗？

## 五、成功标准质量标准

| 等级 | 表现 | 示例 |
|---|---|---|
| **初步** | 有预期成果描述，但不够详细，缺乏具体数据指标和时间节点 | "学生的数学能力有所提高" |
| **发展中** | 较为具体，有基本衡量指标和时间节点，但不够详细 | "大部分学生学期末成绩比学期初有所提高" |
| **有效** | 具体、明确、可衡量，有具体数据描述和明确时间节点 | "下次区抽测中，85%的学生数学至少提升10分，含后30%群体" |
| **卓越** | 具体且灵活适应变化，可监测可评估，可适时调整行动 | "下次区抽测平均分提高至少10%，90%学生达到合格标准；根据中期数据动态调整辅导策略" |

**设定成功标准的讨论提示**：

- 我们目前在哪里？想要达到什么？
- 我们想要看到哪些改变？
- 行动采用后，学生和教师会有什么不同？
- 对不同学生群体，成功应该是什么样的？
- 我们需要量性数据还是质性数据，还是两者都要？

**发现是否达到标准的讨论提示**：

- 问卷、焦点访谈、课堂观察、作业督查、区域调研、考试结果、表现性评价——哪种方式最适合这个行动？
- 不同时间段期望看到哪些改变？

## 六、复盘与评估框架

### 复盘的四个步骤

1. **收集数据**：定期收集与行动相关的数据（教师培训反馈、学生学习成果等）
2. **评估效果**：根据成功标准，对目标达成情况进行评估
3. **总结反思**：分析成功经验和存在问题，提出改进建议
4. **调整优化**：根据评估结果调整下一阶段行动计划

### 监控节奏设计

| 频次 | 内容 | 方式 |
|---|---|---|
| **周会** | 检查进展，课堂观察，教师反馈 | 教师例会汇报+分享案例 |
| **月度会议** | 评估策略有效性，必要时调整 | 管理会议讨论重大问题 |
| **季度总结** | 全面评估实施效果 | 专题会议+数据分析 |

## 七、行动策略质量标准

### 目标设定质量

| 等级 | 表现 |
|---|---|
| **初步** | 有愿景但过于简单笼统，缺乏具体性和深度 |
| **发展中** | 有一定深度，能传达给内部群体，但具体目标可操作性和测量标准不够 |
| **有效** | 连贯清晰，目标有明确可操作性和测量标准，能传达给内外部群体 |
| **卓越** | 简明连贯，含清晰实施步骤和时间表，能传达给所有相关方尤其学生 |

### 行动步骤质量

| 等级 | 表现 |
|---|---|
| **初步** | 有基本步骤但缺乏明确指向性，没有分层，缺乏与目标对应的细节 |
| **发展中** | 与目标有一定对应，有具体策略但缺乏实际考虑，没有分层和改进措施 |
| **有效** | 详细且针对性强，与目标匹配，有具体策略，考虑不同群体，有改进措施和明确职责 |
| **卓越** | 极为详细且完全与目标同步，高度灵活适应变化，有分层策略，有持续改进措施 |

### 实施策略质量

| 等级 | 表现 |
|---|---|
| **初步** | 模糊，无清晰时间线，责任不明确，缺乏预案 |
| **发展中** | 开始考虑资源分配和时间线，但可持续性和适应性不够 |
| **有效** | 全面，有可监控里程碑，详尽责任分配，有应对措施，与日常会议系统结合 |
| **卓越** | 极为详尽，充分考虑实施限制，时间/资源/风险评估合理，有应对方案和可持续机制 |

## 八、改进目标质量标准

当教师从证据链分析中转化出Goal时，用此标准检验目标质量。

### 好目标的五个特征

| 特征 | 检验问题 | 红旗信号 |
|---|---|---|
| **有证据根基** | 目标来源于证据链分析吗？ | 拍脑袋或跟风想出来的 |
| **指向学生变化** | 最终指向学生学习或体验的改善吗？ | 只说教师做什么，不说学生会怎样 |
| **具体而非空泛** | 比"提高教学质量"更具体吗？ | "提高学校整体办学水平"——无法指导行动 |
| **可拆解** | 后续能拆成Objective和行动吗？ | 目标太大太广，无法落地 |
| **与愿景一致** | 与学校长期发展方向对齐吗？ | 与学校战略脱节，只为应付评估 |

### 目标表述脚手架

"通过___（改进行动/策略），我们希望在未来___（时间框架）内，使___（学生群体）在___（具体维度）方面的表现从___（当前状态）提升到___（期望状态）。"

## 十、优先等级排序讨论提示

当教师有多个待改进方向需要选择优先Goal时：

- "如果我们今年不关注这个方向，会发生什么？"
- "为什么相较于明年，我们必须今年就关注这个方面？"
- "如果区域再对我们进行评价，这个方面的改进会影响评价结果吗？"
- "关注这个方向会帮助我们达到长期战略目标吗？"
