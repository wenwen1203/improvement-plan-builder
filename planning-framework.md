# 改进计划制定参考框架 / Reference Framework for Improvement Plan Development

本文件包含各规划阶段的详细质量标准、模板和讨论提示，由 SKILL.md 按需引用。

This document contains detailed quality standards, templates, and discussion prompts for each planning stage, referenced by SKILL.md as needed.

---

## 一、改进计划关键要素 / I. Key Elements of an Improvement Plan

一个完整的改进计划包含以下八个要素：

A complete improvement plan comprises the following eight elements:

| 要素 / Element | 说明 / Description |
|---|---|
| **目标（Goal）** | 学校希望通过改进最终达成的长期结果或总体方向<br>The long-term outcome or overarching direction the school aims to achieve through the improvement effort |
| **具体目标（Objective）** | 为实现Goal需达成的具体、可衡量的短期成果<br>Specific, measurable short-term outcomes required to achieve the Goal |
| **行动 / Action** | 为实现每个Objective所采取的具体步骤<br>Concrete steps taken to accomplish each Objective |
| **时间 / Timeline** | 每个行动的实施时间，含开始和结束日期<br>Implementation period for each action, including start and end dates |
| **负责人 / Person Responsible** | 负责每个行动执行和落实的具体人员<br>Specific personnel accountable for executing and following through on each action |
| **资源需求 / Resource Requirements** | 实现每个行动所需的资源（资金、设备、培训、外部支持等）<br>Resources needed to carry out each action (funding, equipment, training, external support, etc.) |
| **成功标准 / Success Criteria** | 衡量行动和目标达成情况的标准或指标<br>Standards or indicators for measuring the achievement of actions and objectives |
| **复盘与评估 / Review and Evaluation** | 对行动实施效果的总结和评估方式<br>Methods for summarizing and assessing the impact of implemented actions |

## 二、逻辑拆解路径 / II. Logic Decomposition Pathway

### 从Goal到Objective / From Goal to Objective

Goal提供总体方向，Objective将其分解为具体的、可操作的步骤。

The Goal provides the overarching direction; Objectives break it down into specific, actionable steps.

**拆解角度参考 / Suggested Decomposition Angles**（不是每个都需要，选2-4个最相关的 / not all are needed — select the 2–4 most relevant）：

- **教师能力建设 / Teacher Capacity Building**：教师需要掌握什么新策略？（如培训、教研、课堂实践）/ What new strategies do teachers need to master? (e.g., training, collaborative research, classroom practice)
- **课堂实践改变 / Changes in Classroom Practice**：课堂里需要发生什么不同？（如教学设计、评价方式、学习活动）/ What needs to be different in the classroom? (e.g., instructional design, assessment approaches, learning activities)
- **学生支持体系 / Student Support Systems**：学生需要获得什么额外支持？（如个性化辅导、学习资源）/ What additional support do students need? (e.g., individualized tutoring, learning resources)
- **家校协同 / Home-School Collaboration**：家庭层面需要配合什么？（如家长讲座、家庭学习指导）/ What coordination is needed on the family side? (e.g., parent workshops, home learning guidance)
- **资源与制度建设 / Resource and Institutional Development**：学校层面需要提供什么保障？（如资源包开发、制度流程）/ What safeguards does the school need to provide? (e.g., resource kit development, policies and procedures)

**Objective的写法要求 / Writing Requirements for Objectives**：

- 具体且短期：有明确的时间框架和结果 / Specific and short-term: with a clear timeframe and expected outcomes
- 可衡量：包含具体的数字或可观察的变化 / Measurable: including concrete numbers or observable changes
- 直接服务于Goal：每个Objective都应该能回答"它怎么帮助实现Goal？" / Directly serves the Goal: each Objective should be able to answer "How does it help achieve the Goal?"

### 从Objective到行动 / From Objective to Action

**行动设计讨论提示 / Discussion Prompts for Action Design**：

- 我们是否需要实施新的策略或框架？ / Do we need to implement new strategies or frameworks?
- 我们是否需要创建新的政策或指导？ / Do we need to create new policies or guidelines?
- 我们是否需要购买或开发新的资源？ / Do we need to purchase or develop new resources?
- 我们是否需要提供教师培训？ / Do we need to provide teacher training?
- 我们是否需要在教师专业发展中增加新活动？ / Do we need to add new activities to teacher professional development?
- 我们是否需要来自专家的建议？ / Do we need advice from external experts?

**行动筛选讨论提示 / Discussion Prompts for Action Screening**：

- 谁将负责实施？（思考谁有相关专业能力） / Who will be responsible for implementation? (Consider who has the relevant expertise)
- 在负责教师的时间里，这些行动都能完成吗？ / Can these actions be completed within the responsible teacher's available time?
- 是否有其他教师有能力分担？ / Are there other teachers capable of sharing the workload?
- 哪些行动对达成目标是**至关重要**的？ / Which actions are **critical** to achieving the objective?
- 如果不采取某个行动，我们是否仍然能达成目标？ / If we skip a particular action, can we still reach the objective?

**行动之间的依赖关系 / Dependencies Between Actions**：

- 有些行动必须先完成才能启动下一个 / Some actions must be completed before the next one can begin
- 有些行动可以并行推进 / Some actions can proceed in parallel
- 排序时要考虑学期节奏和教师工作负荷 / Consider the semester rhythm and teacher workload when sequencing

## 三、时间规划讨论提示 / III. Discussion Prompts for Timeline Planning

对每个行动，引导教师思考：

For each action, guide teachers to consider:

- 这个行动是否已经是教师的日常工作职责？ / Is this action already part of the teacher's regular job responsibilities?
- 如果是一次性任务，预计花费多少小时？ / If it is a one-off task, how many hours is it expected to take?
- 如果是周期性任务（每周/每学期），每次花费多少时间？ / If it is a recurring task (weekly/per semester), how much time does each instance require?
- 需要多人参与吗？涉及多少人？ / Does it require multiple participants? How many people are involved?
- 行动之间有依赖关系吗？必须先完成哪个？ / Are there dependencies between actions? Which must be completed first?

---

## 四、资源需求讨论提示 / IV. Discussion Prompts for Resource Planning

对每个行动，引导教师思考：

For each action, guide teachers to consider:

- 是否需要外部专家支持？提供什么类型？ / Is external expert support needed? What type?
- 是否需要特定的培训材料或技术平台？ / Are specific training materials or technology platforms required?
- 是否需要额外的场地或设备？ / Are additional venues or equipment needed?
- 是否需要骨干教师的协助？ / Is assistance from lead teachers required?
- 学校现有资源能覆盖吗？需要申请额外预算吗？ / Can existing school resources cover this? Is an additional budget application needed?

## 五、成功标准质量标准 / V. Quality Standards for Success Criteria

| 等级 / Level | 表现 / Performance | 示例 / Example |
|---|---|---|
| **初步 / Emerging** | 有预期成果描述，但不够详细，缺乏具体数据指标和时间节点<br>Describes expected outcomes but lacks detail, specific data indicators, and time targets | "学生的数学能力有所提高"<br>"Students' math ability has improved" |
| **发展中 / Developing** | 较为具体，有基本衡量指标和时间节点，但不够详细<br>More specific with basic metrics and time targets, but still insufficiently detailed | "大部分学生学期末成绩比学期初有所提高"<br>"Most students show improved scores at end of semester compared to the beginning" |
| **有效 / Effective** | 具体、明确、可衡量，有具体数据描述和明确时间节点<br>Specific, clear, and measurable with concrete data descriptions and defined time targets | "下次区抽测中，85%的学生数学至少提升10分，含后30%群体"<br>"In the next district assessment, 85% of students improve math scores by at least 10 points, including the bottom 30% cohort" |
| **卓越 / Exemplary** | 具体且灵活适应变化，可监测可评估，可适时调整行动<br>Specific yet adaptable, monitorable and evaluable, allowing timely action adjustments | "下次区抽测平均分提高至少10%，90%学生达到合格标准；根据中期数据动态调整辅导策略"<br>"In the next district assessment, raise the average score by at least 10% with 90% of students reaching proficiency; dynamically adjust tutoring strategies based on mid-term data" |

**设定成功标准的讨论提示 / Discussion Prompts for Setting Success Criteria**：

- 我们目前在哪里？想要达到什么？ / Where are we now? Where do we want to get to?
- 我们想要看到哪些改变？ / What changes do we want to see?
- 行动采用后，学生和教师会有什么不同？ / What will be different for students and teachers after the action is implemented?
- 对不同学生群体，成功应该是什么样的？ / What should success look like for different student groups?
- 我们需要量性数据还是质性数据，还是两者都要？ / Do we need quantitative data, qualitative data, or both?

**发现是否达到标准的讨论提示 / Discussion Prompts for Determining Whether Standards Are Met**：

- 问卷、焦点访谈、课堂观察、作业督查、区域调研、考试结果、表现性评价——哪种方式最适合这个行动？ / Questionnaires, focus groups, classroom observations, assignment audits, district surveys, exam results, performance assessments — which method best fits this action?
- 不同时间段期望看到哪些改变？ / What changes do we expect to see at different points in time?

## 六、复盘与评估框架 / VI. Review and Evaluation Framework

### 复盘的四个步骤 / Four Steps of Review

1. **收集数据 / Collect Data**：定期收集与行动相关的数据（教师培训反馈、学生学习成果等）/ Regularly collect data related to actions (teacher training feedback, student learning outcomes, etc.)
2. **评估效果 / Assess Impact**：根据成功标准，对目标达成情况进行评估 / Evaluate objective achievement against the success criteria
3. **总结反思 / Summarize and Reflect**：分析成功经验和存在问题，提出改进建议 / Analyze successes and existing issues, and propose recommendations for improvement
4. **调整优化 / Adjust and Optimize**：根据评估结果调整下一阶段行动计划 / Refine the next phase of the action plan based on evaluation findings

### 监控节奏设计 / Monitoring Cadence Design

| 频次 / Frequency | 内容 / Content | 方式 / Method |
|---|---|---|
| **周会 / Weekly Meetings** | 检查进展，课堂观察，教师反馈<br>Check progress, classroom observations, teacher feedback | 教师例会汇报+分享案例<br>Staff meeting reports + case sharing |
| **月度会议 / Monthly Meetings** | 评估策略有效性，必要时调整<br>Assess strategy effectiveness, adjust as needed | 管理会议讨论重大问题<br>Leadership meetings to discuss key issues |
| **季度总结 / Quarterly Reviews** | 全面评估实施效果<br>Comprehensive evaluation of implementation impact | 专题会议+数据分析<br>Dedicated meetings + data analysis |

## 七、行动策略质量标准 / VII. Quality Standards for Action Strategies

### 目标设定质量 / Quality of Goal Setting

| 等级 / Level | 表现 / Performance |
|---|---|
| **初步 / Emerging** | 有愿景但过于简单笼统，缺乏具体性和深度<br>Has a vision but is overly simplistic and vague, lacking specificity and depth |
| **发展中 / Developing** | 有一定深度，能传达给内部群体，但具体目标可操作性和测量标准不够<br>Has some depth and can be communicated internally, but specific objectives lack actionability and measurable standards |
| **有效 / Effective** | 连贯清晰，目标有明确可操作性和测量标准，能传达给内外部群体<br>Coherent and clear; objectives have clear actionability and measurable standards; can be communicated to both internal and external stakeholders |
| **卓越 / Exemplary** | 简明连贯，含清晰实施步骤和时间表，能传达给所有相关方尤其学生<br>Concise and coherent, with clear implementation steps and timelines; can be communicated to all stakeholders, especially students |

### 行动步骤质量 / Quality of Action Steps

| 等级 / Level | 表现 / Performance |
|---|---|
| **初步 / Emerging** | 有基本步骤但缺乏明确指向性，没有分层，缺乏与目标对应的细节<br>Has basic steps but lacks clear direction, tiering, and details aligned with objectives |
| **发展中 / Developing** | 与目标有一定对应，有具体策略但缺乏实际考虑，没有分层和改进措施<br>Partially aligned with objectives; has specific strategies but lacks practical considerations, tiering, and improvement measures |
| **有效 / Effective** | 详细且针对性强，与目标匹配，有具体策略，考虑不同群体，有改进措施和明确职责<br>Detailed and targeted; aligned with objectives; includes specific strategies, considers diverse groups, and has improvement measures with clear responsibilities |
| **卓越 / Exemplary** | 极为详细且完全与目标同步，高度灵活适应变化，有分层策略，有持续改进措施<br>Highly detailed and fully synchronized with objectives; highly adaptable to change; includes tiered strategies and continuous improvement measures |

### 实施策略质量 / Quality of Implementation Strategies

| 等级 / Level | 表现 / Performance |
|---|---|
| **初步 / Emerging** | 模糊，无清晰时间线，责任不明确，缺乏预案<br>Vague; no clear timeline, unclear responsibilities, lacks contingency plans |
| **发展中 / Developing** | 开始考虑资源分配和时间线，但可持续性和适应性不够<br>Begins to consider resource allocation and timelines, but sustainability and adaptability are insufficient |
| **有效 / Effective** | 全面，有可监控里程碑，详尽责任分配，有应对措施，与日常会议系统结合<br>Comprehensive; has monitorable milestones, thorough responsibility allocation, response measures, and integration with regular meeting systems |
| **卓越 / Exemplary** | 极为详尽，充分考虑实施限制，时间/资源/风险评估合理，有应对方案和可持续机制<br>Highly thorough; fully accounts for implementation constraints; realistic time/resource/risk assessment; includes contingency plans and sustainable mechanisms |

## 八、改进目标质量标准 / VIII. Quality Standards for Improvement Goals

当教师从证据链分析中转化出Goal时，用此标准检验目标质量。

When teachers derive a Goal from evidence chain analysis, use this standard to verify goal quality.

### 好目标的五个特征 / Five Characteristics of a Good Goal

| 特征 / Characteristic | 检验问题 / Check Question | 红旗信号 / Red Flag |
|---|---|---|
| **有证据根基 / Evidence-Based** | 目标来源于证据链分析吗？<br>Is the goal derived from evidence chain analysis? | 拍脑袋或跟风想出来的<br>Conceived on a whim or by following trends |
| **指向学生变化 / Student-Oriented** | 最终指向学生学习或体验的改善吗？<br>Does it ultimately point to improving student learning or experience? | 只说教师做什么，不说学生会怎样<br>Only describes what teachers will do, not what will change for students |
| **具体而非空泛 / Specific, Not Vague** | 比"提高教学质量"更具体吗？<br>Is it more specific than "improve teaching quality"? | "提高学校整体办学水平"——无法指导行动<br>"Raise the overall standard of the school" — too vague to guide action |
| **可拆解 / Decomposable** | 后续能拆成Objective和行动吗？<br>Can it be broken down into Objectives and actions? | 目标太大太广，无法落地<br>Goal is too broad to be actionable |
| **与愿景一致 / Aligned with Vision** | 与学校长期发展方向对齐吗？<br>Is it aligned with the school's long-term direction? | 与学校战略脱节，只为应付评估<br>Disconnected from school strategy; created merely to satisfy evaluation requirements |

### 目标表述脚手架 / Goal Statement Scaffold

"通过___（改进行动/策略），我们希望在未来___（时间框架）内，使___（学生群体）在___（具体维度）方面的表现从___（当前状态）提升到___（期望状态）。"

"Through ___ (improvement action/strategy), we aim to improve the performance of ___ (student group) in ___ (specific dimension) from ___ (current state) to ___ (desired state) within ___ (timeframe)."

## 十、优先等级排序讨论提示 / X. Discussion Prompts for Priority Ranking

当教师有多个待改进方向需要选择优先Goal时：

When teachers have multiple areas for improvement and need to select a priority Goal:

- "如果我们今年不关注这个方向，会发生什么？" / "What will happen if we do not focus on this area this year?"
- "为什么相较于明年，我们必须今年就关注这个方面？" / "Why must we address this area this year rather than next year?"
- "如果区域再对我们进行评价，这个方面的改进会影响评价结果吗？" / "If the district evaluates us again, will improvement in this area affect the evaluation results?"
- "关注这个方向会帮助我们达到长期战略目标吗？" / "Will focusing on this area help us achieve our long-term strategic goals?"
